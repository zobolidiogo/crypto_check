Sobre o crypto.check

O crypto.check é uma plataforma web desenvolvida para simular investimentos em criptomoedas utilizando dados reais de mercado.

O projeto foi criado como Final Project do CS50x (Harvard University) com foco em desenvolvimento full-stack, integração de APIs e visualização de dados.

A aplicação utiliza Flask no backend, PostgreSQL e Supabase como banco de dados em nuvem e APIs públicas para atualização de preços em tempo real.

Usuários podem criar contas, comprar e vender criptomoedas virtualmente, acompanhar histórico de transações e visualizar gráficos interativos de mercado.

Tecnologias utilizadas:
Python • Flask • PostgreSQL • Supabase • JavaScript • Chart.js
